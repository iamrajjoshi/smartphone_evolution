How to Use

  • Click on the buttons to switch between different features (Battery, Memory, Storage, Camera) or view the overall trend (All).
  • Use the dropdown menu to filter by brand (not applicable for the "All" view).
  • Hover over points to see detailed information about each smartphone.
  • Observe how smartphone features have evolved over time and compare different brands.
  • The annotation highlights the highest value for each feature or the peak year for phone releases.
